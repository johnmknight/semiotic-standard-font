**Pipeline:**
```
Hand-drawn PNG  →  step1_extract.py   →  trace_all.py    →  step5_compose.py   →  step6_fix_winding.py  →  build-font-direct.js
(from john/)       (crop + B&W)          (OpenCV trace)     (+ Cobb frame)         (nonzero winding)        (TTF / WOFF / WOFF2)
```
